Декомпозиция проблемы:
1. Файл *start_task* (функция start_programm()) - запускает работу программы, предлагает варианты выполнения действий
2. Функция data_input() - осуществляет добавление нового абонента в блокнот
3. Функция get_all_data() - выводит все данные на экран
4. Функция get_data() - функция-скрипт - табло выбора действий
5. Функция search_data () - пользовательский интерфейс пользователю по поиску информации
6. Функция search_argument(index, argument) - считывает всю информацию из блокнота в переменную и ищет искомые данные, если количество найденных абонентов больше 1, то эти данные ищет в:
7. Функция search_list (use_list) - осуществляет сужение искомого абонента по дополнительным параметра до 1, чтобы в последствии предложить пользователю иные действия с данными (здесь не происходит считывание информации из блокнота. Функция работает с данными, пришедшими из функции см.6)
8. Функция edit_data(use_string) - редактирует данные искомого абонента до подтверждения внесения изменений в блокнот
9. Модуль delete_data для использования функции удаления найденного абонента
10. Модуль insert_data для использования функции по вставке абонента согласно алфавиту и нумерации

  Примечание:
  1. Данные программа принимает с пробелами, которые удаляет и любым регистром, который приводит к заглавному
  2. Удаление и редактирование данных осуществляется только после нахождения конкретной искомой позиции
  3. Добавление и удаление информации осуществляет с перезаписью информации в блокноте, с соответствующим изменением в списках
  4. Программа зацикливается до того момента, пока пользователь не указывает позицию для выхода
  5. Некоторые функции имеют цикличную ссылку друг на друга, всвязи с чем не имелось возможность разделения кода на большее количество модулей
